#### Proyectos

**[AutoPilot](https://github.com/fsaldivar-dev/AutoPilot)** — Automatización iOS desde macOS. Sin XCUITest, sin servidor, sin dependencias. Un binario de 311KB que controla el Simulador via APIs de accesibilidad nativas.

**[kuilo](https://github.com/fsaldivar-dev/kuilo)** — IDE de documentación técnica local-first. Electron + React + Tiptap + MCP. Pensado para equipos de una persona.

**[dagger-swift](https://github.com/fsaldivar-dev/dagger-swift)** — Inyección de dependencias resuelta en compile-time. Property wrappers (`@Singleton`, `@Provider`) + módulos tipados. Si compila, las dependencias están resueltas.

**[sajarubox-mcp](https://github.com/fsaldivar-dev/sajarubox-mcp)** — MCP server que expone reglas de negocio, arquitectura y contexto de sprint para sesiones de Claude Code.

**[CameraAppMac](https://github.com/fsaldivar-dev/CameraAppMac)** — Cámara flotante para macOS. Preview circular, sin grabación, sin almacenamiento.

**[experimental-annotation-swift](https://github.com/fsaldivar-dev/experimental-annotation-swift)** — Property wrappers para validación de atributos en Swift. 92% code coverage, CI/CD, publicado en CocoaPods.
